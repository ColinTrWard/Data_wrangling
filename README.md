# Data Wrangling with Python - Diabetes & Healthcare Expenditure

A data wrangling project that gathers, assesses, cleans, and analyzes two 
real-world datasets to investigate the relationship between diabetes prevalence 
and healthcare expenditure across different countries.

## Research Question

**Is there a relationship between diabetes prevalence and healthcare expenditure 
as a percentage of GDP across different countries?**

## Overview

This project demonstrates a full data wrangling pipeline using two real-world 
datasets. The data is gathered, assessed for quality issues, cleaned, combined, 
and then analyzed to answer the research question through visualizations.

## Tech Stack

- Python 3
- pandas
- seaborn
- matplotlib
- Jupyter Notebook

## Key Concepts Covered

- Data gathering from multiple sources
- Programmatic and visual data assessment
- Data quality and structural issue identification
- Data cleaning strategies
- Combining and storing datasets
- Exploratory data analysis (EDA)
- Research-driven visualization

## Datasets

- **Diabetes Dataset** — contains health metrics including BMI and diabetes 
prevalence across countries
- **Health Expenditure Dataset** — contains healthcare spending as a percentage 
of GDP across countries

## Key Findings

- A scatter plot of diabetes prevalence vs health expenditure % GDP reveals 
the potential correlation between healthcare spending and diabetes rates
- A box plot of BMI distributions across low, medium, and high expenditure 
groups shows how healthcare spending relates to population health metrics

## Visualizations

1. **Scatter Plot** — Diabetes Prevalence vs Health Expenditure % GDP
2. **Box Plot** — BMI Distribution by Health Expenditure Group

## How to Run

1. Clone the repo
2. Install dependencies: `pip install pandas seaborn matplotlib`
3. Open `Data_Wrangling_Project_Starter.ipynb` in Jupyter
4. Run all cells top to bottom

## Reflection

Future analysis could incorporate additional health metrics such as average 
age or income levels across countries to reveal deeper socioeconomic influences 
on diabetes prevalence and provide a more comprehensive view of public health 
drivers globally.
